== Description ==

Add the Proliker button to your blog.
This plugin does require that data is loaded from an external site (proliker.com).
For more informations: http://proliker.com

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you want to change the settings, click Settings->The Proliker Button and change the settings there.
